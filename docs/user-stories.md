# FuelSaver User Stories

This document outlines the core user needs addressed by the FuelSaver
application.

## Price Discovery & Map Interactions

### 1. Visualizing Local Prices

**As a** driver, **I want to** see fuel prices pinned to their actual locations
on an interactive map, **so that** I can easily identify the cheapest stations
along my current route or near my destination.

### 2. Contextual Area Metrics

**As a** savvy shopper, **I want** the "Area" average metrics to update
automatically as I pan and zoom the map, **so that** I can see how prices
compare within the specific neighborhood I am currently viewing.

## Filtering & Personalization

### 3. Fuel Type Selection

**As a** vehicle owner, **I want to** filter the map and list to show only the
fuel types my car uses (e.g., U91, E10), **so that** I don't get distracted by
irrelevant price data.

### 4. Brand Loyalty

**As a** member of fuel reward programs, **I want to** filter stations by brand
name, **so that** I can prioritize stations where I can use my discount vouchers
or earn points.

### 5. Persistent Preferences

**As a** frequent user, **I want** my fuel type and brand selections to be
remembered across sessions, **so that** the app is already tailored to my needs
every time I open it.

## Comparison & Contextual Metrics

### 6. Price Benchmarking

**As a** budget-conscious user, **I want to** see at a glance how much a
station's price deviates from the State average and the local Area average, **so
that** I can instantly know if a station is offering a "good" or "bad" deal
relative to the market.

### 7. Detailed Station Insights

**As a** user, **I want to** click on a map marker to see a detailed breakdown
of all available fuel types and prices for that specific station, **so that** I
can verify all options before deciding where to fill up.

## User Experience & Accessibility

### 8. Map vs. List View

**As a** user, **I want to** toggle between a map view and a sorted list view,
**so that** I can choose the format that best helps me find the cheapest price
(visual vs. tabular).

### 9. Eye Comfort

**As a** night-time driver, **I want** the application to support dark mode
(both for the UI and the map tiles), **so that** I can use the app safely
without being blinded by a bright screen.
